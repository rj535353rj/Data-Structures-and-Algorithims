Dependency Manager
=================

Dependency Manager is a program to ensure that their are no repeated definitions of the functions and structs generated by the macros in this project.
Every header defining a concrete data structure included in this project should have a matching file with the .dp extention.
The syntax for a basic .dp file is as follows:

```.dp
#dp
path: "path-to-main-folder"
inc: "path-from-main-folder-to-target"
inc: "path-from-main-folder-to-other-target" "argument-one" "argument-two"
```

for example:

```dp
#dp
path: "Documents/DSA"
inc: "Number/Integer"
inc: "Collection/List/ArrayList" "Integer"
```

for datastructures that are generated by macros, you can use additional arguments to define its subtypes.
